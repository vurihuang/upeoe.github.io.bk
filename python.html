      pip 安装包使用阿里云镜像加速

      ~/.pip/pip.conf

      [global]
index-url = http://mirrors.aliyun.com/pypi/simple/
[install]

      Virtualenv

      $ pip3 install virtualenv

      Usage

      # 创建环境容器，文件夹名为 `ENV`
$ virtualenv ENV

$ ls
bin		include		lib		pip-selfcheck.json

# 激活虚拟开发环境，现在安装库以及使用的都是以当前文件夹/容器优先
# 并且会在命令行左侧提示符(prompt)增加一项 `env name`，表明你处在哪一个虚拟环境容器中
$ cd ENV && source bin/activate


# 可以看到，环境变量会优先从 `ENV/bin` 下查找
$ echo $PATH
/Users/[username]/workspace/python/ENV/bin:/user/local/bin:/usr/bin:/bin:/bin...

$ python --version
Python 3.6.5

$ pip --version
pip 10.0.1

$ which python
/Users/[username]/workspace/python/ENV/bin/python

$ which pip
/Users/[username]/workspace/python/ENV/bin/pip

# 不做配置的话，默认会安装如下内容，包括复制的一份 `Python`
$ pip list

pip        10.0.1
setuptools 39.2.0
wheel      0.31.1

# 退出当前容器
$ deactivate

      IPython

      IPython 是一个可交互的 python shell 解释器，支持语法高亮、 tab 补全、直接执行系统命令、保存历史记录等等功能。

      $ pip3 install ipython
